จะสร้าง web app 
เก็บค่าส่วนกลางหมู่บ้าน 
โดยในแต่ละเดือน จะมีการ อับโหลด สลิป (รูป ) กับ ใส่ข้อความ เช่น ค่าส่วนของของเดือน พค บ้านเลขที่ 39/50 ช่วยออกแบบ ระบบ 

Front end 
- ชื่อ app "PFS" ย่อมาจาก PraiFah smart 
- เน้นตัวใหญ่ๆ ดูในมือถือชัด 
- หน้าบ้าน สวย ใช้งานง่าย มี ปุ่มกด อับโหลดรูป ตรงเลขที่บ้าน ให้สามารถเลือก เลขที่กับ เดือนได้ 
- ระบบ ตรวจเช็คยอดจากสลิป 300 บาทไหม 
- ถ้าอับโหลดรูป +เลือก เดือน กับ บ้านเลขที่ แล้ว ให้ ขึ้น เรียบร้อย ถ้าไม่ครบให้มี aleart ตามหัวข้อ 
- มีระบบ แจ้งถ้าเลือก เดือน ซ้ำ 
- มีปุ่ม กด แสดง ย้อนหลัง ว่าจ่ายแล้วกี่เดือน 
- มีระบบ แจ้งเตือนหากเลยวัน ที่จ่าย 
- Tracking ชื่อคนโอน เพื่อตรวจสอบ ว่า ตรงกับ เลขที่บ้าน ไหม 

Back end 
- Upload ข้อมูล บน cloud ทั้งรูปและ ข้อความ แยก ตามบ้านเลขที่ 
- ถ้าผู้ใช้ต้องการดูข้อมูล สามารถแสดงได้ตลอด (ดูได้เฉพาะบ้านตัวเอง) 
- มี id pass สำหรับ admin โหลดข้อมูล ทั้งหมด เป็น exel หรืออื่นๆ 
- ตั้งได้ ให้ โหลดข้อมูล เป็น สัปดาห์ หรือ เดือน เพื่อ backup data กันหาย 
- กำหนดวันจ่าย ได้ 
- ระบบ ฐานข้อมูล แต่ละบ้าน กับ ชื่อ คนในบ้าน
